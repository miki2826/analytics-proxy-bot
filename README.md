analytics-proxy-bot
================

## Get Started
- Run `npm i` 
- Provide the following environment variables:
 - `FACEBOOK_PAGE_ID`
 - `FACEBOOK_ACCESS_TOKEN`
 - `FACEBOOK_VERIFY_TOKEN`
 - `DASHBOT_ACCESS_TOKEN`
- Run `npm start`
 
To run with docker please insert the above environment variables to the `Dockerfile`
